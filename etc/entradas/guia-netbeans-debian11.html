ARTÍCULO BAJO CONSTRUCCIÓN

Guía de descarga e instalación del IDE NetBeans en Debian 11 "Bullseye"

Aunque soy fiel a Sublime Text (me gusta mucho cómo funciona y es bastante fácil trabajar con el repositorio de GitHub), estoy aprendiendo bases de datos con MySQL y un poquito de Java y esto ya se me queda corto.

Un buen colega, me recomendó Eclipse y he estado leyendo que es perfecto para Java y se puede preparar para trabajar con SQL, sin embargo es un poco tedioso hacerlo. Entre las características de NetBeans se encuentra que es mucho más sencillo hacer el proceso anterior porque tiene incorporado el soporte para bases de datos, de modo, que he decidido probar con éste a ver qué tal.

1 - Descargar e instalar el runtime y el kit de desarrollo de Java

Para que todo funcione bien, necesitamos el JRE (la máquina virtual de Java para poder ejecutar los programas escritos en dicho lenguaje) y el JDK (el conjunto de herramientas necesarias para escribir código de Java, compilarlo...)

Abrimos la consola de comandos y ejecutamos la orden:
sudo apt install openjdk-17-jdk openjdk-17-jre -y

2 - Descargar el script de instalacion de Apache NetBeans

Una vez termine el proceso, entramos en este enlace

Si os fijais la siguiente imagen, he rodeado con un rectángulo rojo el enlace que contiene el script.
Algo que NO me ha gustado demasiado es que debes hacer un WGET para poderd descargártelo y, si sabes de Linux, sabes cómo se obtiene ese archivo pero alguien que empieza de cero no tiene por qué saber eso. Vuelvo a mi recurrente frase de "luego nos quejamos de que Linux no triunfa en el mercado de escritorio.

De todos modos, no os penséis que esta guía es muy "muy user-friendly": no he podido encontrar manera menos desagradable que explicarte para qué sirve cada comando y dártelo todo pasito a pasito y masticadito para que no te caigas de espaldas si es tu primera vez con Linux.

IMAGEN- copiar enlace de postimage.

En resumidas cuentas: lo que debeis hacer es:

  1. Copiar el enlace del rectángulo rojo en el portapapeles (simplemente haz click derecho y selecciona Copiar dirección de enlace).
  2. Abre la consola de comandos.
  3. Crea una carpeta con el comando mkdir netbeans, después escribe wget enlace-que-has-copiado (El enlace se pega con Ctrl + Shift + V · ¡Recuerda dejar un espacio entre wget y el enlace).
  4. El script es algo pesadito y tarda un poco en descargarse, pero, una vez haya finalizado, cambia a la carpeta Netbeans y añade permisos de ejecución con cd netbeans && sudo chmod +x presiona-tabulador
  5. Por último, ejecuta el script: sudo sh ./presiona-tabulador

El resto de la instalación transcurre de forma normal, ya que este script se parece más a un instalador gráfico de Windows.

Por defecto, la instalación incluye:

  • El editor de código.
  • Las ediciones Standard y Enterprise de Java.
  • Paquete de HTML5, JS (JavaScript) y PHP.

Todo ello, redondeando, no llega a ocupar los 800 MB en disco, así que no nos preocupemos por eso.
Cabe decir que yo he dejado todo lo del instalador por defecto.

Hacemos click en Finish y ya hemos terminado, ahora, solamente queda abrir el programa y abrirse sin ningún problema.
A mi me ha funcionado a la primera.
